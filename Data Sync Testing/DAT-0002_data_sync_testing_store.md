## **DAT-0002:** Data Sync Testing - Data Store

> **Summary:** Verify that the system will firstly store all changes in the local storage or in cache storage. <br>

**Preconditions:**

- User must be connected to the internet
- User must be logged in

Scenario 1

| \#  | Step                  | Expected Behavior                                                    |
| --- | --------------------- | -------------------------------------------------------------------- |
| 1   | Vist the mobile app.  | Verify that the login page is shown.                                 |
| 2   | Perform any activity. | Verify that the activity is performed and displayed, if appliclable. |

**Post-conditions:**

- The data is stored locally or in the cache.
